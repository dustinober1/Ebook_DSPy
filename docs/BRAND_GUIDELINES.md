# DSPy Ebook - Brand Guidelines

## Overview
This document defines the brand identity and visual guidelines for the DSPy: A Practical Guide ebook.

## Brand Personality
- **Professional**: Technical accuracy and authority
- **Educational**: Clear, approachable, and supportive
- **Modern**: Clean design with contemporary aesthetics
- **Practical**: Focus on real-world applications

## Color Palette

### Primary Colors
- **DSPy Blue**: #1E88E5 (Primary brand color)
- **Deep Blue**: #1565C0 (Dark variant)
- **Light Blue**: #42A5F5 (Light variant)

### Secondary Colors
- **Success Green**: #4CAF50
- **Warning Orange**: #FF9800
- **Error Red**: #F44336
- **Info Teal**: #00BCD4

### Neutral Colors
- **Dark Gray**: #263238
- **Medium Gray**: #546E7A
- **Light Gray**: #CFD8DC
- **White**: #FFFFFF
- **Off-white**: #F5F5F5

## Typography

### Headings
- **Font**: Inter (web), Source Sans Pro (PDF/EPUB)
- **Weights**: 600 (H1), 500 (H2), 400 (H3+)
- **Line Height**: 1.2

### Body Text
- **Font**: Inter (web), Source Sans Pro (PDF/EPUB)
- **Weight**: 400
- **Size**: 16px (web), 11pt (PDF)
- **Line Height**: 1.6

### Code
- **Font**: JetBrains Mono (web), Source Code Pro (PDF/EPUB)
- **Weight**: 400
- **Size**: 14px (web), 10pt (PDF)
- **Background**: #F6F8FA

## Logo Usage

### DSPy Logo
The DSPy logo should be used consistently across all materials:
- Primary logo: Full color on light backgrounds
- Secondary logo: White on dark backgrounds
- Minimum size: 120px width for web, 2cm width for print

### Logo Placement
- **Book Cover**: Top center or top left
- **Website**: Top left navigation
- **PDF**: Header or title page
- **Marketing**: Prominent placement, clear visibility

## Cover Design Specifications

### Print Cover (PDF)
- **Size**: 6" x 9" (15.24cm x 22.86cm)
- **Resolution**: 300 DPI
- **Format**: PDF, CMYK color
- **Bleed**: 0.125" (3.175mm) on all sides

### Digital Cover (Web)
- **Size**: 1400 x 2100 pixels
- **Resolution**: 72 PPI
- **Format**: PNG or JPG, RGB color
- **Aspect Ratio**: 2:3

### Cover Elements
1. **Title**: "DSPy: A Practical Guide"
2. **Subtitle**: "Programming Foundation Models"
3. **Author**: "Dustin Ober"
4. **Visual Elements**:
   - Abstract neural network/LLM visualization
   - Code snippets or syntax elements
   - Clean, modern design
5. **Color Scheme**: Primary blue with accent colors
6. **Typography**: Bold, readable, hierarchical

## Visual Elements

### Icons
- Style: Line art, 2px stroke
- Color: DSPy Blue or Dark Gray
- Size: Consistent within context

### Diagrams
- Style: Clean, minimal, technical
- Colors: Blue palette with accent colors
- Labels: Clear, readable, consistent typography

### Screenshots
- Style: High contrast, minimal UI
- Border: 1px solid #E1E4E8
- Shadow: Subtle drop shadow for depth

## Layout Guidelines

### Grid System
- **Web**: 12-column grid, max-width 1200px
- **Print**: Baseline grid, margins 1.5cm

### Spacing
- **Base unit**: 8px (web), 6pt (print)
- **Component padding**: 16px (web), 12pt (print)
- **Section spacing**: 48px (web), 24pt (print)

### Hierarchy
1. Page title (H1): 36px/24pt
2. Chapter title (H2): 28px/20pt
3. Section title (H3): 22px/16pt
4. Subsection (H4): 18px/14pt
5. Body text: 16px/11pt
6. Caption: 14px/9pt

## Usage Guidelines

### Do's
- ✅ Use official brand colors
- ✅ Maintain consistent typography
- ✅ Ensure accessibility (WCAG AA)
- ✅ Keep designs clean and uncluttered
- ✅ Use high-quality images

### Don'ts
- ❌ Modify the logo proportions
- ❌ Use unapproved colors
- ❌ Overcrowd designs
- ❌ Use low-resolution images
- ❌ Break accessibility guidelines

## File Formats

### For Web
- Images: WebP, PNG (transparent), JPG (photos)
- Icons: SVG
- Fonts: WOFF2

### For Print
- Images: TIFF, PNG (300 DPI)
- Final document: PDF/X-1a

## Brand Assets Location
All brand assets are stored in `/assets/brand/`:
- `/logo/` - Logo files in various formats
- `/covers/` - Cover designs and templates
- `/icons/` - Custom icon set
- `/templates/` - Design templates
- `/guidelines/` - Additional guidelines
