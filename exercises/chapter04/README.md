# Chapter 4: Evaluation Exercises

This directory contains practice exercises for Chapter 4: Evaluation.

## Contents

- `problems.md` - Exercise descriptions and requirements
- `solutions/` - Complete solutions with explanations

## Exercises

| Exercise | Topic | Difficulty | Time |
|----------|-------|------------|------|
| 1 | Building a Quality Dataset | Intermediate | 30 min |
| 2 | Designing a Comprehensive Metric | Intermediate-Advanced | 45 min |
| 3 | Comprehensive Evaluation Pipeline | Intermediate-Advanced | 45 min |
| 4 | Preventing Data Leakage | Advanced | 60 min |
| 5 | Evaluation Report Generator | Advanced | 60 min |

## Getting Started

1. Read the exercise descriptions in `problems.md`
2. Try to implement each solution independently
3. Use the hints if you get stuck
4. Compare your solution with `solutions/solutions.py`

## Prerequisites

- Working DSPy installation
- Completed Chapter 4 reading
- Reviewed examples in `examples/chapter04/`

## Running Solutions

```bash
# Test all solutions
cd exercises/chapter04
python solutions/solutions.py
```

## Learning Objectives

After completing these exercises, you should be able to:

- Create balanced, representative evaluation datasets
- Design multi-aspect custom metrics
- Run comprehensive evaluations with detailed analysis
- Prevent data leakage in evaluation pipelines
- Generate stakeholder-ready evaluation reports
