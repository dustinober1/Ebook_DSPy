Chapter 5

InPars+: Advanced Synthetic Data Generation

Using DSPy and Contrastive Preference Optimization (CPO) to generate high-quality synthetic data for training information retrieval models.

Introduction

InPars+ is an enhanced framework for generating synthetic queries to train Information Retrieval (IR) systems. It improves upon the original InPars method by integrating DSPy prompt optimization and Contrastive Preference Optimization (CPO).

Key Innovations

  • CPO Fine-tuning: Models are trained to distinguish between high-quality and low-quality queries.
  • Dynamic Prompting: DSPy modifies the query generation prompt based on the specific document type and context.
  • Reduced Filtering: Higher quality generation means less data needs to be thrown away.

Architecture

1. CPO Fine-tuned Generator

A language model is fine-tuned to prefer better queries using a contrastive loss function.

2. DSPy Dynamic Optimizer

Instead of a static prompt like "Generate a query for this document", DSPy optimizes the instructions.
