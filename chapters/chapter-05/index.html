üìä The Expected Performance Maximization Framework

At the core of DSPy's optimization philosophy is the Expected Performance Maximization Framework:

Goal: maximize E[metric(program(parameters), data)]

Where:
- Œ∏ = program parameters (instructions, demos, weights)
- D = data distribution
- f = evaluation metric
- P_Œ∏ = parameterized program

Instead of manually crafting prompts and hoping for good results, DSPy treats prompt and model optimization as a principled optimization problem.

‚ö° Quick Example

import dspy
from dspy.teleprompt import BootstrapFewShot

# Traditional approach: Manual prompting
prompt = "Answer the question carefully."  # Hope it works...

# DSPy approach: Expected performance maximization
optimizer = dspy.BootstrapFewShot(
    metric=answer_accuracy,     # Define what success means
    max_bootstrapped_demos=4    # DSPy finds optimal examples
)

# Compile automatically optimizes the program
optimized_program = optimizer.compile(
    program,
    trainset=training_examples
)

# Result: Maximized expected performance!
