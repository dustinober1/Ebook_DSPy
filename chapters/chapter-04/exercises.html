Exercise 1 ‚≠ê Beginner

Write Your First Metric

Task: Create metrics for a Q&A system.

Requirements:

  • Create an exact_match metric (case-insensitive)
  • Create a contains_answer metric (answer appears in prediction)
  • Create a word_f1 metric (F1 score of word overlap)
  • Test all three on sample predictions
# Test your metrics on these examples:
examples = [
    {"expected": "Paris", "predicted": "Paris"},
    {"expected": "Paris", "predicted": "The answer is Paris"},
    {"expected": "William Shakespeare", "predicted": "Shakespeare"},
]

# Your metrics should return:
# exact_match: [True, False, False]
# contains_answer: [True, True, True]
# word_f1: [1.0, 0.33, 0.5]
Exercise 2 ‚≠ê Beginner

Create a Training Dataset

Task: Build a training set for a topic classifier.

Requirements:

  • Create 15 dspy.Example instances for topic classification
  • Include 5 examples each for: "technology", "sports", "politics"
  • Use .with_inputs("text") to mark inputs
  • Split into train (10) and dev (5) sets

Expected structure:

example = dspy.Example(
    text="The latest smartphone features an AI chip...",
    topic="technology"
).with_inputs("text")
Exercise 3 ‚≠ê‚≠ê Intermediate

Basic BootstrapFewShot Optimization

Task: Optimize a sentiment classifier using BootstrapFewShot.

Requirements:

  • Create a sentiment analysis signature (positive/negative/neutral)
  • Build a training set of at least 10 examples
  • Define a matching metric
  • Run BootstrapFewShot optimization
  • Compare baseline vs. optimized accuracy
  • Save the optimized module

Report these metrics:

  • Baseline accuracy on dev set
  • Optimized accuracy on dev set
  • Percentage improvement
Exercise 4 ‚≠ê‚≠ê Intermediate

LM-as-Judge Metric

Task: Create an LM-based metric for evaluating summaries.

Requirements:

  • Create a signature for summary evaluation:
    • Inputs: original_text, summary
    • Outputs: accuracy (0-1), completeness (0-1), conciseness (0-1)
  • Create a metric function that:
    • Calls the LM judge
    • Returns the average of all three scores
  • Test on 3 different summary examples
