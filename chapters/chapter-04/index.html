The DSPy Optimization Paradigm

Instead of manually crafting prompts, DSPy uses a fundamentally different approach:

‚ùå Traditional Approach

# Manually write prompts
prompt = """You are an expert at...
Think step by step...
Here are some examples:
Example 1: ...
Example 2: ...
Now solve: {question}"""

# Trial and error to improve
# Hope it works on new data
Manual iteration Time-consuming Inconsistent results

‚úÖ DSPy Approach

# Define what you want
module = dspy.ChainOfThought(MySignature)

# Provide training data
trainset = [dspy.Example(...), ...]

# Let DSPy optimize
optimized = optimizer.compile(
    module,
    trainset=trainset
)
Data-driven Automatic Reproducible
