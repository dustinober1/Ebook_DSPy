Why Signatures Matter

Signatures are the core abstraction in DSPy. They separate what you want from how it's accomplished:

‚ùå Traditional Prompting

# You write the HOW
prompt = """
You are a helpful assistant.
Please summarize the following text
in 2-3 sentences. Be concise and
capture the main points.

Text: {text}

Summary:
"""
‚ö†Ô∏è Fragile instructions ‚ö†Ô∏è Hard to optimize

‚úÖ DSPy Signature

# You specify the WHAT
class Summarize(dspy.Signature):
    """Summarize text concisely."""
    text: str = dspy.InputField()
    summary: str = dspy.OutputField(
        desc="2-3 sentence summary"
    )
‚úÖ Declarative and clean ‚úÖ Auto-optimizable
